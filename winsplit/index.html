下载和安装

WinSplit是我正在使用的一款分屏软件。但是目前官网已经停止了更新，转而开发商业收费的MaxTo：

WinSplit Revolution has unfortunately been discontinued, and there are no new updates coming to this great product.

不过通过百度依然可以下载到WinSplit，大小不到1.5M，运行时只占8M内存，支持32和64位的Windows系统。

安装分为Classic和Portable，选啥自己看着办吧。

7 8 9，如1为左上、5为最大化，也可使用虚拟键盘（一个悬浮小窗口）		Ctrl + Alt + Numeric Pad [1-9]



Automatic placement
使窗口快速适应到自定义的布局（位置和大小），在Options中删除
Ctrl + Alt + Numeric Pad 0


Window fusion
窗口融合，控制两个窗口水平或竖直分屏显示，可调整分屏比例
Ctrl + Alt + F


Mosaic
马赛克化，现有窗口自动布局，如“目”、“田”型
Ctrl + Alt + M


Close all window
关闭所有窗口（建议禁用）
Ctrl + Alt + C


Active window tools
当前视窗工具，用于控制当前窗口是否始终在最上层、调节窗口透明度、截图
Ctrl + Alt + T


Toggle Always on Top
控制当前窗口是否始终在最上层
Ctrl + Alt + O


Move to Left screen
移动当前窗口到左屏（适合多屏工作者）
Ctrl + Left


Move to Right screen
移动当前窗口到右屏（适合多屏工作者）
Ctrl + Right


Drag’N’Go
鼠标拖动窗口到固定布局
Ctrl + Alt + 鼠标拖动窗口
